C++ 排序 (sort) 常见类型与模板

发表于2022-12-14|更新于2022-12-21|学习笔记
|字数总计:1.7k|阅读时长:8分钟|阅读量:|评论数:

挺久之前的笔记（还不如叫模板）了

为什么浏览器会把这个当成挪威语页面啊

排序 Sort

选择排序 Selection Sort

每 i 次找出第 i 小的元素，然后与第 i 个元素交换位置。

不稳定

最好时间、平均时间、最坏时间 $O(n^2)$

冒泡排序 Bubble Sort

每次检查相邻两个元素，若前面的元素与后面的元素满足给定的排序条件就交换。当没有相邻元素需要交换时，排序完成。

不稳定

最好时间 $O(n)$

平均时间、最坏时间 $O(n^2)$

插入排序 Insertion Sort

从未排序数组中选择一个插入到已排序数组的正确位置。

稳定

最好时间 $O(n)$

平均时间、最坏时间 $O(n^2)$

计数排序 Counting Sort

使用一个额外的数组记录待排序数组中相同元素的个数，之后根据额外数组排序待排序数组。

稳定

时间 $O(n+w)$ 其中 $w$ 代表待排序数组元素的取值域大小

基数排序 Radix Sort

将待排序的数组拆分成 k 个关键字，逐个比较关键字。

稳定

时间 $O(kn+\sum_{i=1}^kw_i)$

快速排序 Quicksort

通过分治的方式来将一个数组排序。

不稳定

最好时间、平均时间 $O(n\ log\ n)$

最坏时间 $O(n^2)$

归并排序 Merge Sort

通过给数组分段之后合并来排序。

稳定

最好时间、平均时间、最坏时间 $O(n\ log\ n)$

堆排序 Heapsort

堆排序是建立在堆上的选择排序。

不稳定

最好时间、平均时间、最坏时间 $O(n\ log\ n)$

桶排序 Bucket sort

桶排序是建立在桶上的排序方法，适用于待排序数据值域较大但分布比较均匀的情况。

稳定

最好时间、平均时间 $O(n+n^2/k+k)$

最坏时间 $O(n^2)$

希尔排序 Shell sort

希尔排序是插入排序的一种改进版本。

不稳定

最好时间 $O(n)$

平均时间 $O(n^{3/2})$

最坏时间 $O(n\ log^2\ n)$

锦标赛排序 Tournament sort

两两比较，胜者进入下一轮。

类似于树。

不稳定

最好时间、平均时间、最坏时间 $O(n\ log\ n)$
